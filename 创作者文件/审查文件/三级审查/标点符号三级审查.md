# 标点审查方法论 · 三步法

> 一级审查：GB/T 15834—2011（详见 `../一级审查/GB_T_15834_标点符号用法.md`）
> 二级审查：项目追加约束（详见 `../二级审查/标点符号二级审查.md`）
> 三级审查：本文，三步法操作流程 + 扫描顺序
> 数字依据：GB/T 15835—2011（详见 `../一级审查/GB_T_15835_数字用法.md`）
> 用字依据：GF 1001-2001（详见 `../一级审查/GF_1001_2001_异形词整理表.md`）

核心原则：**每个符号都必须有不可替代的存在理由**。三步测试依次执行——删去、替换、重构。前一步通过即停，不必走后续步骤。

---

## 通用三步流程

```
发现一个标点符号
    │
    ▼
第一步：删去
  删掉该符号，朗读。
  通顺 → 删掉符号（结束）
  不通 → 进入第二步
    │
    ▼
第二步：替换
  按符号选择替换方向，朗读。
   句号 → 逗号
   分号 → 逗号 或 句号
   逗号 → 空
   顿号 → 空
   破折号／省略号（引号外）→ 逗号 或 ：（冒号仅限解释说明）
  通顺 → 改为此符号（结束）
  不通 → 进入第三步
    │
    ▼
第三步：重构
  调整语序或表达方式，从根本上消除对这个符号的需求。
  通顺 → 改写句子（结束）
  不通 → 保留原符号
```

---

## 一、逗号

**第一步·删去：** 删掉 `，`，朗读。通顺 → 删。

`她端起茶杯，啜了一口。`
→ 删：`她端起茶杯啜了一口。` → 通顺 → 删逗号 ✓

`他走进房间，关上门，坐下。`
→ 删：`他走进房间关上门坐下。` → 通顺 → 两处全删 ✓

**第二步·替换：** 删去后不通，说明需要停顿。降级已是最轻符号，无需替换，直接保留。

**第三步·重构：** 如果删去不通、但句子本身笨重，考虑换表达方式来消除逗号需求。

`灵力耗尽，她靠在墙边。`
→ 删：`灵力耗尽她靠在墙边。` → 不通，但句子可以重构 → `她灵力耗尽，靠在墙边。` → 删去第一处逗号后：`她灵力耗尽靠在墙边。` → 通顺 → 重构后删逗号 ✓

---

## 二、句号

**第一步·删去：** 删掉 `。`，直接合并前后句。通顺 → 删句号（两句合一）。

`副科长。战术思维如棋盘般精密。`
→ 删：`副科长战术思维如棋盘般精密。` → 不通，"副科长"是独立标签 → 进入第二步

**第二步·替换：** 将 `。` 换为 `，`。通顺 → 改逗号。

`副科长。战术思维如棋盘般精密。`
→ 替换：`副科长，战术思维如棋盘般精密。` → 通顺 → 改逗号 ✓

**硬止条件（替换为逗号后仍不通 → 保留句号）：**
- 段落末尾
- 前后主语不同，且后句是全新的语义起点
- 后句为感叹/疑问句

**第三步·重构：** 替换后仍不通，考虑重新组织前后两句的表达。

`她拒绝交出抄本，离开了协会。齿轮给了她第三条路。`
→ 替换：`…离开了协会，齿轮给了她…` → 主语切换，不通 → 重构：`她拒绝交出抄本，离开了协会，随后齿轮给了她第三条路。` → 加入"随后"连接，逗号通顺 ✓

---

## 三、分号

**第一步·删去：** 删掉 `；`，直接连接。通顺 → 删。

**第二步·替换：** 换为 `，` 或 `。`。

**互换测试作为辅助判断：** 分号前后两个分句互换位置。语义不变 → 并列关系 → 分号可留。语义改变 → 非并列 → 必须替换。

`月：冷酷；果断；保护欲强；算计 / 明：怯懦；勤勉；无意识；温和`
→ 互换：`明：怯懦、勤勉；月：冷酷、果断。` → 语义不变 → 分号正确 ✓

**第三步·重构：** 非并列但分句复杂时，重写为两句独立的句子。

---

## 四、破折号 / 省略号

**第一步·删去：** 在引号（`""`）内 → 合法，保留。在引号外 → 违禁，删掉。通顺 → 删。

`她沉默了——然后转身离开。`
→ 删：`她沉默了然后转身离开。` → 不通 → 进入第二步

**第二步·替换：** 换为 `，` 或 `：`。
→ 替换：`她沉默了，然后转身离开。` → 通顺 → 改逗号 ✓

**第三步·重构：** 换标点仍生硬时，重写句子避开需要破折号的表达。

`她的灵力回路——构造——允许她以灵力构造任何形态的实体。`
→ 删/替换：`她的灵力回路，构造，允许她…` → 两个逗号略显拥挤 → 重构：`她的灵力回路名为构造，允许她以灵力构造任何形态的实体。` → 加入"名为"消除同位语标点需求 ✓

---

## 五、顿号

**第一步·删去：** 删掉 `、`。通顺 → 删。粘连不清 → 保留。

`白衬衫、黑裤、短靴。`
→ 删：`白衬衫黑裤短靴。` → 三项粘连 → 保留顿号 ✓

**第二步·替换：** 换为 `，`（不同层级的列举）。

**第三步·重构：** 列举项过多时，拆为短句分行。

---

## 六、引号

唯一合法引号：`""`（英文直双引号）。`''` `''` `「」` `『』` → 一律替换。

---

## 七、扫描顺序

逐文件扫描顺序，从机械到判断层层递进，每步只依赖前步结果，不回头：

1. 引号格式：非 `"` (U+0022) → 统一替换。引号未统一前，后续所有"引号内外"判断不可靠。
2. `——` `……` `！` `？`在引号外 → 违禁，三步法处理。硬违禁先清场，若降级为逗号则流入后续步骤。
3. `，` `、` → 删去测试。最轻的操作，且上一步可能产出新标点，在此一并处理。
4. `；` → 互换测试。机械判断，上下文已理顺。
5. `。` → 句号判例（二级审查 §七）。最吃判断力的放最后，此时标点环境已清理干净，误判率最低。
