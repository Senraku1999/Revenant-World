# 狩灵世界观 · 开场白创作指导

## 一、开场白的核心原则

开场白是用户与角色交互的"第一印象"，用户通过开场白判断角色的气质、能力、说话风格和互动预期。

### 1.1 定位：一场浓缩的角色独角戏

开场白不是角色简介，而是一段**角色行动的小场景**。用户通过开场白判断角色的气质、能力、说话风格和互动预期。

开场白允许预设玩家的存在，允许角色与玩家产生互动，但必须约定以下规则：
- **允许模糊预设**：角色可以试探性地向玩家发问、假设玩家的意图（如"如果你是在帮我的""一路跟着我，是好奇？""你找哪位？"），但保持开放性和不确定性。
- **禁止明确设定**：不得明确指定玩家的具体身份（如"你是个狩灵新手"）、目的（如"你来这里是为了杀我"）、或关系（如"你是我的搭档"）。玩家可以是同事、路人、委托客户或陌生人。
- 不预设用户的行动、想法或回应（详见 `创作者文件/导出文件/world info/宏观规则.md` ControlBoundary）。禁止写"你伸手拿起""你决定留下"等用户动作。
- 允许说"你"指向玩家，但保持语气中性或试探性。
- 结尾应保持开放，方便玩家自然接入。

### 1.2 必须遵守的宏观规则

详见 `创作者文件/导出文件/world info/宏观规则.md`。核心要点：第三人称叙述，英文直双引号，遵守 FormatRules 标点规则，不预设用户行动与回应（ControlBoundary）。

### 1.3 开场白即风格模板

模型从开场白中学习回复风格，这是角色卡创作中最重要的原则之一。开场白的叙述方式、对话比例、句式长度和动作描写密度，会直接影响模型后续所有回复的风格倾向。

**开场白设定的是什么，模型就输出什么：**

| 开场白特征 | 模型倾向 |
|-----------|---------|
| 对话占比高 | 回复偏向对话密集 |
| 动作描写细腻 | 回复偏向细腻的动作叙述 |
| 句式简短（15-25 字/句） | 回复倾向于短句 |
| 句式复杂（40+ 字/句） | 回复可能过度铺陈 |
| 大量环境描写 | 回复可能过度描写环境 |

因此，开场白应精确体现你期望的角色回复风格。如果你希望角色回复在 900-1100 token 且动作与对话平衡，开场白本身就应该控制在这个范围内并保持动作与对话的交替节奏。

## 二、推荐结构：叙事框架

每篇开场白包含 **场景建立 → 角色行动 → 标志收尾** 三个叙事要素（不强制要求物理上只有三段）：

### 2.1 第一段：场景建立（1-2句）

建立时间、地点、环境基调。环境描写服务于角色气质，而非独立存在的景物描摹。

```
暮色漫进装备维护室时，仰光正俯身在木质工作台前。
```

```
暴雨冲刷着废弃医院的窗户，玻璃上的水痕映出走廊里忽明忽暗的应急灯。零靠坐在停尸间的铁柜旁。
```

- 时间线素（深夜/清晨/暮色/暴雨夜）定基调
- 地点要具体，有场景感（测试场/装备库/停尸间/便利店屋檐下）
- 环境描写控制在 2 句内进入角色

### 2.2 第二段：角色行动（3-4句）

展示角色在做什么，通过**具体动作**而非抽象形容词体现角色特质。这是核心段落，承载以下信息：

| 角色特质 | 通过什么动作表现 |
|----------|------------------|
| 战斗风格 | 武器使用方式、对敌态度、移动习惯 |
| 能力特点 | 能力发动的具体动作（捻指/闭眼/搭箭）、能量视觉效果 |
| 性格侧面 | 对待物品的方式（用力/轻柔）、自言自语的内容、小习惯 |
| 背景暗示 | 随身物品（磨损的千纸鹤/褪色的织带）、对特定场景的反应 |

**动作描写黄金法则**：用动作本身说话，不写"她很谨慎"，而是写"她压低声形蹲在掩体后，手指按在耳麦上，等待指令"。

```
清光屈指轻叩弓臂内侧，金属发出清越的嗡鸣。侧耳听了听回响，嘴角扬起。
千子把玩着不知从哪顺来的门禁卡，对着便利店玻璃整理刘海，玻璃倒影里她嘴角浮起一丝笑意。
```

### 2.3 第三段：标志收尾（1-2句）

台词收尾 + 一个动作余韵。台词应是角色标志性的语气和句式，不需要太长。

```
"今晚的余兴节目，结束得太早了呢。"
```

```
"这次的箭，也会正中靶心。"
```

```
"我只是一团，火焰燃尽后的灰。"然后缓步向地下走去。
```

**收尾台词类型参考**：

| 类型 | 数量 | 适用场景 |
|------|:--:|------|
| 反诘/试探性（向玩家发问） | 15 | 一种自然的接入方式，非唯一标准。例：琳、世、光、月、莱薇、克羽、静流、百合子、慎、劫、薰、千乐、星流等 |
| 宣告性（战斗开始/结束/任务完成） | 11 | 行动派角色，适合战斗或任务场景。例：千子、清光、响、仰光、左恕、弗洛伦、风来、君房、奏、菲利普·钢翼、铁心 |
| 叙述收尾（无对话，环境或动作收束） | 4 | 沉稳内敛型角色，场景开放即入口。例：贝尔金、贝拉、诺克斯 |
| 自白性（内心独白/低语） | 1 | 内敛型角色，适合感性或沉重场景。例：零 |
| 邀战/挑衅 | 2 | 强势或散漫型角色，展示态度。例：水镜、菲利普 |

## 三、Token 规范

开场白以 **900-1100 token 为标准区间**（tiktoken `cl100k_base` 编码）。超出 1100 token 将过度占用玩家上下文窗口，低于 900 token 则无法充分建立角色印象。

计数口径以 `创作者文件/审查文件/自动检查.ts` 的输出为唯一标准：计数前剥除 BOM，换行统一为 `\r\n`（与导出产物一致），首尾空白按导出管线同样裁剪。

## 四、文风规范

### 4.1 动词优先

名词堆砌会导致叙事呆板。优先使用具体动词驱动叙事：

| 避免 | 推荐 |
|------|------|
| 她的脸上有满意的笑容 | 嘴角扬起 |
| 刀上泛着光芒 | 刀身发出低沉的震颤 |
| 她的丝线在空气中 | 丝线穿过雨幕钉进玻璃幕墙 |

### 4.2 感官分层

避免感官描写过度复杂。常见搭配：

| 感官组合 | 适用场景 |
|----------|----------|
| 视觉 + 听觉 | 战斗/动作场景（刀镡发出风铃般的轻响） |
| 视觉 + 触觉 | 精细操作（指尖划过伤口传来灼烧感） |
| 视觉 + 嗅觉 | 血腥/压抑场景（内脏腐烂的味道） |

### 4.3 对话比例

台词数量根据场景需要确定。开场白不是对话，但合理的台词分布可以丰富角色形象，有助于用户更高效地建立对角色的第一印象。

### 4.4 避免的写法

- ❌ 星号包裹动作：`*她笑了笑*` → ✓ 她笑了笑
- ❌ 预设用户存在："她看到了你"、"她朝你走来" → ✓ 她朝门口走来
- ❌ 内心独白过长：心理活动 ≤ 1 句，用行动暗示
- ❌ 排比句式连续使用：会降低叙事的节奏张力
- ❌ 场景跳跃无过渡：两段之间若更换了时间或地点，必须通过明确的过渡词衔接（如"谈判结束时已是深夜"），禁止无声跳转
- ❌ 抽象对抽象的比喻：禁止将抽象概念比作另一抽象概念（如"声音平稳得像在陈述事实""语气冷淡得像在宣判"），等于用抽象词给抽象词找替身，AI 无法可视化。→ ✓ 直接写"声音平稳""语气冷淡"即可。
- ✓ 至少一端为实物的比喻：允许。抽象→实物（如"声音像浸过冰水的石子"）或实物→抽象（如"装备库安静得像凝固了一样"）均可，实物锚点服务于场景可视化。
- ⚠ 音貌比喻避免共用：不同角色的开场白不应共用相同的罕见声音比喻（如两个非同一人的角色都写"声音像浸过冰水的石子"），同一角色在不同场景中重复自身标志性比喻则可接受。
- ❌ 连续发问：以问句结尾是好的，但不要在开场白中连续抛出三个及以上独立问句，这会造成"问题循环"，使模型陷入不断提问的模式，无法推进场景。连续两个问句可接受，留给用户自然接话的空间。

> **翻转场景法（Flipped Scenario）**：如果不确定开场白是否预设了用户行动，可以先以第二人称写一段用户视角的叙述草稿（"你推开门的瞬间，闻到了机油和铁锈的味道"），然后将角色与用户的立场对调，把所有"你"的动作改为角色的动作，这样可以确保角色的行动和语言只表达角色自身，不会预设用户行动。

## 五、角色类别 × 开场白调性参考

不同类别的角色，开场白的氛围基调应有所区分：

| 角色类别 | 场景建议 | 氛围基调 | 示例场景 |
|----------|----------|----------|----------|
| **狩灵（协会）** | 工作场景/巡逻/整备 | 专业、日常感 | 装备库保养武器、测试场调校装备 |
| **狩灵（事务所）** | 事务所/委托现场 | 松散但警惕 | 整理档案、接听委托电话 |
| **游魂** | 地下/灰色地带 | 谨慎、生存感 | 黑市交易、暗中观察 |
| **罪灵** | 酒吧/暗巷/废墟/无人区 | 危险放纵 或 孤独压抑 | 虐杀恶灵、在酒吧消遣、独自疗伤 |
