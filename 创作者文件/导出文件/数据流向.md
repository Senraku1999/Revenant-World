# 数据流向

## 两条独立管线

本项目有两条互不交叉的数据管线：

---

## 管线一：角色卡导出

```
{角色}.json + {角色}简介.md + {角色}开场白.md
        │
        └── 拼装 ──→ {角色}角色卡.json（SillyTavern 导入用）
```

**说明：** 详见 `角色卡导出规范.md`。

---

## 管线二：世界书构建

```
创作者文件/导出文件/world info/*.md   创作者文件/导出文件/角色关系网/{分组}/{角色}.json
  （基础规则条目）                      （角色补充词条）
        │                                    │
        └──────────┬─────────────────────────┘
                   │
                   ▼
              完整世界书 JSON
         （基础规则 + 角色词条）
```

详见 `世界书构建规范.md`。

---

## 角色关系网的来源

```
{角色}.json（角色卡）
      │
      └── 提取扁平化 ──→ 创作者文件/导出文件/角色关系网/{分组}/{角色}.json
```

详见 `世界书构建规范.md`。
