> JSON 字段中出现本表"禁止"列所列形式时，必须替换为"强制全称"列的中文全称。

## 五、扩展规则

以下为跨字段通用规则：

| 规则 | 说明 |
|------|------|
| 狩灵协会X科 | 任何位置不得省略"狩灵协会"前缀，不得用英文 department X 替代 |
| 组织名中英一致 | char_faction 用中文全称；char_status、char_relationships 等英文描述字段中，同义组织也必须用中文全称 |
| 评级后缀 | JSON 罪灵/异常评级须带"级"后缀；简介评级匹配对应中文格式 |
| 灵力≠灵能 | 详见 `狩灵角色卡创作指导.md` §3.1 完整区分表（体系/分类→灵能，能量/流动/构筑→灵力）。典型错误：灵能凝结的箭矢→灵力凝结的箭矢，灵力者→灵能者 |
