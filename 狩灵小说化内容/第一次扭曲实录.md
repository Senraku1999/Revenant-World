# 第一次扭曲实录

林野是一个没人记得住的狩灵考核落选者。灵力资质过弱，灵视范围不足三米，身体素质比普通上班族还差。考核官在评语栏写了一句“建议放弃灵能相关职业”，林野签了字，回到东京一家中型文具公司继续做他的设计员。那是平成三十年的春天。

他在公司被叫做“小林”。他的工位在203室靠窗的位置，桌上常年堆着未拆封的快递盒和半满的咖啡杯。妻子两年前病逝，女儿由岳母在埼玉老家带着。他每个月汇去一半工资，剩下的刚好覆盖房贷。同事们知道他家里出了事，但没有人问细节。林野也不说。他每天早上九点打卡，晚上七点离开，午餐在便利店解决，周末去埼玉看女儿。他的生活在任何人看来都是一条平直下降的线，只是斜率缓到没人注意。

同年十一月开始，同事注意到一些细微的变化。林野不再在午休时聊天。他盯着电脑屏幕发呆的时间越来越长，有时一份设计稿在显示器上停了整个下午，鼠标没有移动过一厘米。他开始在打印纸的边缘画细密的网格线，用红笔在每个格子里写一个从未见过的符号。同事问他符号是什么意思，他抬头茫然地看了对方的脸，像是确认了某种存在。他的身体也开始消瘦，不是因为吃得太少，而是因为他的灵力正在以无法被检测的方式在体内积累。五科后来的灵力波形分析报告显示，他在这一时期已进入前兆期：灵力波形高度紊乱，间歇性的灵视幻听持续干扰感知。他从未主动寻求帮助。他不认为自己的症状与灵力有关，考核早已判定他的灵力“不足以构成风险”，而他接受了这个结论。他不知道残余的灵力仍在以无法被检测的方式积累。

前兆期持续了三个月。

突变发生在一个二月的午后。阳光斜斜切进203室的百叶窗，在地面投下菱形光斑。林野坐在堆叠的办公椅上，后颈抵着打印机的冰凉外壳。主管的声音还在他脑子里炸响，“小林，这份报表什么时候弄完？上个月的项目奖金扣光了，这个月的房贷可不能逾期。”他低头看自己的手，曾经握着钢笔画出无数设计稿的指节，正在被某种钝痛扯得变形，皮肤下凸起细密的骨节，像被强行掰弯的铁丝。

打印机突然发出刺耳的嗡鸣。

他的脊椎传来撕裂般的痛。办公桌从四面八方挤压过来，不是他触摸它们，是它们像铁屑被磁石吸引一样，撞进他的身体。实木桌腿扎进腰腹，金属桌角刮破皮肤，血珠顺着扭曲的桌面往下淌。他的头颅被一台显示器从内部顶起，屏幕裂开蛛网状的纹路，露出下面跳动的蓝光，那是他每天早会用的电脑，屏幕上的待办事项还在闪烁，“9：00 部门会议”“11：30 客户对接”，可他的眼前只剩泛着雪花点的世界。左手开始分叉，五根手指延展、变薄、硬化成裁纸剪刀的刃口，每一次开合都有细碎金属碎片从指缝掉出来。他想起自己曾用这把剪刀为女儿裁出星星形状的折纸。右臂正在被复印机的顶盖从内部撑开，开口装置缓缓张开，像一张永远无法闭合的嘴，发出低沉的、充满渴望的嗡鸣。

他的脚开始下沉。地面的瓷砖被下半身的重量挤压碎裂，粉末状的填缝剂从裂缝中簌簌落下，像他积压了三年的焦虑终于找到了出口。所有的抑制，房贷合同上的签名、妻子葬礼上的白花、女儿发烧时他背在背上的体温，在灵力的催化下不再是记忆，而是燃料。显示器屏幕上的蓝光变成刺目的红，那红色顺着脊椎往上窜，烧穿了头皮，烧穿了胸腔，烧穿了所有被压抑的东西。

他张开嘴。不是发出声音，而是让整层楼的结构在他体内共振。堆叠的办公桌像多米诺骨牌般倒塌。打印机吐出最后一格墨，溅在变形的手臂上。阳光从破碎的窗户灌进来，照在他已经不再属于人类的身体上，一个仅有上半身的巨像，头部是碎裂的显示器，左手是裁纸剪刀，右手是复印机开合装置，胸腔由二十三把办公椅的金属支架组成。他的眼睛，显示器雪花屏深处残留的最后一点属于林野的东西，已经没有了迷茫，只有纯粹的、想要毁灭一切的黑暗。

建筑物保安在闻到胶木烧焦的气味后调出了203室的监控画面。画面里整层楼的墙壁正在从内侧向外鼓胀，办公隔间的玻璃隔断一片接一片炸裂，混凝土粉尘从天花板的裂缝中倾泻而下。他拨了警方的电话，说的第一句话是：“有栋楼要塌了。”

警方到场时发现203室所在楼层的结构已经完全变形，走廊被垮塌的天花板堵塞，电梯井里灌满了碎裂的办公家具。任何试图靠近的尝试都会引发更剧烈的结构坍塌。现场指挥官在评估后拨通了狩灵协会的紧急联络号码，警方的标准手册中有一条备注：当现场出现“无法归因于常规物理原因的建筑物结构性损毁”时，上报狩灵协会，不追问理由。

协会对此毫无概念。没有先例，没有手册，没有专门应对这种事的科室。按照官僚体系的标准流程，异常事件自动归入治安科（二科）的管辖范围。一科的调配官员翻了一遍名册，派出了当晚值勤的大阪二科标准巡逻小队，四个人，小队长三阶，其余三人四阶。调配官员在派遣记录上写的理由是“常规异常处理”。他没有见过扭曲。整个协会没有人见过扭曲。

巡逻小队进入大楼后十七分钟，通讯完全中断。事后从残存的灵力痕迹和建筑损毁模式中推断，战斗过程不到四分钟。小队长第一个阵亡：巨像的剪刀左手直接贯穿了他的胸腔，连同他身后的隔断墙一起劈开。第二名队员在试图从侧面接近时被复印机开合部件拦腰夹碎。剩下的两人在撤退中被巨像以纯粹的物理质量碾压，那只由二十三把办公椅支架和数张实木办公桌压缩而成的手臂，单次挥击的力量超过了任何常规恶灵的打击上限。四具尸体，没有一具完整的。他们面对的不是需要技术拆解的复杂异常，而是一台不需要任何技巧、只凭质量和蛮力就能摧毁一切的重型机器。

协会在一小时内升级了响应级别。调令直接发到了当时在总务科（一科）的薰手上。薰在一科还不到一年。

薰到达现场时，巨像已经将办公层的半面墙撞穿。薰在走廊里站了三秒。确认天丛云剑的封印可以被解除，确认解除后八岐大蛇的意志处于可压制状态。第四秒，他拔出了剑。

战斗持续了十一分钟。薰以天丛云剑的正面斩击强行碾压扭曲的灵力场。剪刀左手在第一次交锋中被紫色火焰裹挟的剑身劈断。复印机右手在第二次斩击中被纵向剖开，内部喷出的是数百张扭曲变形的人脸图像，林野的同事们被扭曲吞噬的记忆残片。最后一击落在胸腔正中，二十三把办公椅的金属支架在紫色火焰中同时熔化。巨像的显示器头部在熄灭前闪过了最后一个画面：一个男人在签文件，抬起头，看向屏幕外的薰。然后画面永远消失了。

事后调查确认了两项事实。第一，扭曲实体的破坏力远超其灵力读数。林野的灵力资质在考核时被判定为“过弱”，但突变后的实体仅凭物理蛮力就在四分钟内撕碎了一支标准巡逻小队。一科评级委员会的评估结论是：传统的灵力读数无法预测扭曲实体的实际威胁，评级必须反映事件造成的结构性冲击，四名狩灵阵亡、半栋楼被毁、以及这种威胁类型的全新性，灾厄级。一科科长和副科长在最终文件上签了字，但附加了一份备忘录，指出协会应首先承认这并非“异常事件”，而是一个全新的范畴。

第二项事实是，协会对落选灵能者的后续追踪存在系统性盲区。协会此前默认“考核落选者会回归普通生活，不再构成灵能风险”，林野证明了这个前提的致命缺陷。五科事后调阅了所有考核落选者的档案，发现其中超过四十人的灵力波形在落选后出现了渐进式紊乱。

评级文件签署后不到一周，一科科长向全协会签发了第一份色彩赋予令。色彩：靛蓝。称号：靛蓝封禁。薰在接到令状时沉默了很长时间，不是拒绝，而是面对一个他从未要求过的责任时，那种过度承担责任的人才有的犹豫。他要求科长给他一夜时间。那一夜他独自坐在一科大楼的档案室里，握着天丛云剑的剑柄，在黑暗中与剑对话。次日清晨，他在令状上签了字。

一个月后，狩灵协会第六科，扭曲科，在福冈正式挂牌。薰被任命为首任科长。六科成立之初只有他一个人。
